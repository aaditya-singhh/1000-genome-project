The raw input was a **27 GB VCF file** (`ALL.chrX.phase3_shapeit2_mvncall_integrated_v1c.20130502.genotypes.vcf`) covering **2,504 samples** from 5 super-populations (AFR, AMR, EAS, EUR, SAS).

## Part 1 — x_analysis.ipynb

### What Was Done
- Converted the 27 GB VCF to CSV using chunked streaming (no memory crash)
- Parsed genotypes with sex-aware logic — females are diploid (0|0, 0|1, 1|1), males are hemizygous (0, 1)
- Applied QC filters: FILTER=PASS, QUAL ≥ 30, biallelic only, call rate ≥ 95%
- Computed per-variant allele frequency and call rate across all 2,504 samples
- Computed per-sample missing rate and alternate allele burden
- Computed population-level allele frequencies separately for AFR, AMR, EAS, EUR, SAS
- Identified population-differentiated variants (max − min AF > 0.3)

### Key Findings

| Metric | Value |
|---|---|
| Total variants after QC | **3,437,097** |
| Samples | **2,504** (1,271 female / 1,233 male) |
| Population-differentiated variants (ΔAF > 0.3) | **132,173** |
| Per-sample missing rate | **0%** (complete genotyping) |
| Mean alternate allele burden per sample | **~105,576** |

**Population sample sizes:**
- AFR: 661 &nbsp;|&nbsp; EAS: 504 &nbsp;|&nbsp; EUR: 503 &nbsp;|&nbsp; SAS: 489 &nbsp;|&nbsp; AMR: 347

**Output files:** `chrX_variant_summary_qc.csv`, `chrX_sample_summary_qc.csv`, `chrX_AF_by_population.csv`, `chrX_differentiated_variants.csv`

### 2. Population Differentiation (FST)

**Pairwise FST Matrix:**

|  | AFR | AMR | EAS | EUR | SAS |
|---|---|---|---|---|---|
| AFR | 0 | 0.0821 | **0.1210** | 0.1038 | 0.0943 |
| AMR | 0.0821 | 0 | 0.0401 | 0.0253 | **0.0227** |
| EAS | 0.1210 | 0.0401 | 0 | 0.0752 | 0.0382 |
| EUR | 0.1038 | 0.0253 | 0.0752 | 0 | 0.0289 |
| SAS | 0.0943 | 0.0227 | 0.0382 | 0.0289 | 0 |

**Finding:**
- **Most differentiated:** AFR vs EAS (FST = 0.121) — the largest genetic distance, reflecting deep population separation
- **Least differentiated:** AMR vs SAS (FST = 0.023) — consistent with shared ancestry and gene flow
- These FST values are higher than typical autosomal values, expected for the X chromosome due to its lower effective population size (¾ of autosomes)

**PCA:** PC1 (91.0%) + PC2 (5.5%) = 96.5% of variance explained — AFR separates cleanly from non-African populations on PC1.

### 4. Selection Analysis — Tajima's D

| Population | Tajima's D |
|---|---|
| AFR | -0.185 |
| AMR | -0.171 |
| EAS | -0.177 |
| EUR | -0.160 |
| SAS | -0.174 |

**Finding:** All populations show mildly negative Tajima's D (around -0.17), indicating an excess of rare variants. This is consistent with **population expansion** and/or **purifying selection** acting on the X chromosome — a known biological signal. No population exceeded |D| > 1, so no strong directional or balancing selection was detected.
